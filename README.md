## quickstart


cd twincontracts
```

```bash

yarn install

```

> you might get node-gyp errors, ignore them and run:

```bash

yarn run start

```

> in a second terminal window:

```bash

yarn run chain

```

> in a third terminal window:

```bash

yarn run deploy

```
